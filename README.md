# Day 19-29 Learning Repository

A comprehensive learning repository for mastering GitHub, Git, automation, and community features.

## 📚 What is this?

This repository documents a 29-day learning journey covering:
- Git fundamentals and workflows
- GitHub Issues, Pull Requests, and Projects
- GitHub Actions and automation
- Code quality and testing
- AI-assisted coding (Copilot)
- Community and documentation

## ✨ Features

- ✅ Issue templates for consistent reporting
- ✅ Automated workflows with GitHub Actions
- ✅ Project boards for visualization
- ✅ Milestones for release planning
- ✅ Discussions for community engagement
- ✅ Wiki for internal documentation
- ✅ GitHub Pages for public docs
- ✅ Comprehensive examples

## ⚙️ Workflows

### Issue Automation
- Automatically adds `priority-medium` label to new issues  
- Integrates with GitHub Projects  

### Linting
- Runs on PR creation  
- Checks code quality  
- Prevents merge if failed  

### Testing
- Runs on PR  
- Executes test suite  
- Reports coverage
