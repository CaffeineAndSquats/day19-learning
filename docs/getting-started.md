# Getting Started

## What is this project?

This is a learning project for mastering GitHub workflows, automation, and community features.

## Prerequisites

- Git installed
- GitHub account
- Basic command line knowledge

## Installation

```bash
git clone https://github.com/CaffeineAndSquats/day19-learning.git
cd day19-learning
```

## First Steps

1. Read the main README
2. Check the Wiki for detailed docs
3. Browse the Issues
4. Join the Discussions
5. Create your first issue

## Need Help?

- Check the Wiki
- Ask in Discussions
- Search existing Issues
